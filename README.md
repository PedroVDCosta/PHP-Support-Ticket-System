# PHP-Support-Ticket-System

A demonstration project built in PHP to simulate an IT support ticket system, including form validation, data handling and basic workflow management.

---

## Features

* Create support tickets via form
* Input validation
* Ticket listing
* Search and filtering
* Session-based data storage
* Structured for future database integration

---

## Technologies

* PHP
* HTML / CSS
* Session handling
* Basic MVC-style structure
* PDO (prepared for MySQL integration)

---

## Purpose

This project was developed to demonstrate practical skills in:

* Backend development with PHP
* Form handling and validation
* Data processing and organization
* Building simple operational tools for IT environments

---

## How to Run

```bash
php -S localhost:8000
```

Open in browser:

```
http://localhost:8000
```
